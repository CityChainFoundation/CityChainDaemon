Subtraction(CachePerformanceSnapshot, CachePerformanceSnapshot)

Creates a snapshot based on difference of two performance counter snapshots.

This is used to obtain statistic information about performance of the cache during certain period.

Declaration
public static CachePerformanceSnapshot operator -(CachePerformanceSnapshot end, CachePerformanceSnapshot start)
Parameters
Type Name Description

Older performance counter snapshot.
Returns
Type Description
CachePerformanceSnapshot

Snapshot of the difference between the two performance counter snapshots.
Remarks

The two snapshots should be taken from a single performance counter. Otherwise the start times of the snapshots will be different, which is not allowed.
