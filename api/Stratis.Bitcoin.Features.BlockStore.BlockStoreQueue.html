Class BlockStoreQueue

Saves blocks to the database in batches, removes reorged blocks from the database.

Remarks

The batch is saved when total serialized size of all blocks in a batch reaches Stratis.Bitcoin.Features.BlockStore.BlockStoreQueue.BatchThresholdSizeBytes, or when more than Stratis.Bitcoin.Features.BlockStore.BlockStoreQueue.BatchMaxSaveIntervalSeconds passed since last batch was saved, or when node is shutting down.

When we save new blocks to the database, in case IBlockRepository contains blocks that are no longer a part of our best chain, they are removed from the database.

When block store is being initialized we delete blocks that are not on the best chain.
