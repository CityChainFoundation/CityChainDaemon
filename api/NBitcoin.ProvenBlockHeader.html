Class ProvenBlockHeader

Extension to an existing BlockHeader which is used in PoS to prevent attacker from constructing a fake chain of headers that has more work than the valid chain and attacking a node.

Proven header prevents such an attack by including additional information that can be validated and confirmed whether the header is fake or real.

Remarks

Additional information included into proven header:

Block header signature (Signature), which is signed with the private key which corresponds to coinstake's second output's public key.

Coinstake transaction (Coinstake).

Merkle proof (MerkleProof) that proves the coinstake tx is included in a block that is being represented by the provided header.
