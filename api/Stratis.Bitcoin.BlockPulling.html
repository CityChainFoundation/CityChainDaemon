Namespace Stratis.Bitcoin.BlockPulling

Classes

BlockPuller

Base class for pullers that download blocks from peers.

This must be inherited and the implementing class needs to handle taking blocks off the queue and stalling.

BlockPuller.DownloadedBlock

Description of a block together with its size.

DownloadAssignment

Describes the assigned download tasks.

LookaheadBlockPuller

Puller that is used for fast sync during initial block download (IBD). It implements a strategy of downloading multiple blocks from multiple peers at once, so that IBD is fast enough, but does not consume too many resources.

LookaheadResult

A result from the ILookaheadBlockPuller containing the downloaded block and the of the peer that it came from. Block will be null if the current chain was reorganized.

PullerDownloadAssignments

Implements a strategy for a block puller that needs to download a set of blocks from its connected peers.

PullerDownloadAssignments.PeerInformation

Information about a connected peer.

QualityScore

Implements logic of evaluation of quality of node network peers based on the recent past experience with them with respect to other node's network peers.
