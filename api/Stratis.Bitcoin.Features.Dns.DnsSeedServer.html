Remarks

The DnsFeature object is designed to produce a whitelist of peers from the IPeerAddressManager object which is then periodically formed into a new masterfile instance and applied to the IDnsServer object. The masterfile is swapped for efficiency, rather than applying a merge operation to the existing masterfile, or clearing the existing masterfile and re-adding the peer entries (which could cause some interim DNS resolve requests to fail).
