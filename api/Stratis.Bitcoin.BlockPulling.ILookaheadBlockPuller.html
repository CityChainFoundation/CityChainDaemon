TryGetLookahead(Int32)

Tries to retrieve a block that is in front of the current puller's position by a specific height.

Declaration
Block TryGetLookahead(int count)
Parameters
Type Name Description
System.Int32 count

How many blocks ahead (minus one) should the returned block be ahead of the current puller's position. A value of zero will provide the next block, a value of one will provide a block that is 2 blocks ahead.
Returns
Type Description
Block

The block which height is count+1 higher than current puller's position, or null if such a block is not downloaded or does not exist.
