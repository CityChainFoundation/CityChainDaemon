SaveChanges(IList<UnspentOutputs>, IEnumerable<TxOut[]>, uint256, uint256, Int32, List<RewindData>)

Persists changes to the coinview (with the tip hash oldBlockHash) when a new block (hash nextBlockHash) is added and becomes the new tip of the coinview.

This method is provided (in unspentOutputs parameter) with information about all transactions that are either new or were changed in the new block. It is also provided with information (in ) about the previous state of those transactions (if any), which is used for Rewind() operation.

Declaration
public void SaveChanges(IList<UnspentOutputs> unspentOutputs, IEnumerable<TxOut[]> originalOutputs, uint256 oldBlockHash, uint256 nextBlockHash, int height, List<RewindData> rewindDataList = null)
Parameters
Type Name Description
IList<UnspentOutputs> unspentOutputs

Information about the changes between the old block and the new block. An item in this list represents a list of all outputs for a specific transaction. If a specific output was spent, the output is null.
IEnumerable<TxOut[]> originalOutputs

Information about the previous state of outputs of transactions inside unspentOutputs. If an item here is null, it means that the ouputs are newly created in the new block. If it is not null, it holds information about which outputs of the transaction were previously spent and which were not.
uint256 oldBlockHash

Block hash of the current tip of the coinview.
uint256 nextBlockHash

Block hash of the tip of the coinview after the change is applied.
System.Int32 height

The height of the block.
List<RewindData> rewindDataList

List of rewind data items to be persisted. This should only be used when calling SaveChanges(IList<UnspentOutputs>, IEnumerable<TxOut[]>, uint256, uint256, Int32, List<RewindData>).
