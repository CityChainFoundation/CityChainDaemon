PushBlock(uint256, Block, Int32)

Removes assignments for the block which has been delivered by the peer assigned to it and calls the callback.

Declaration
public void PushBlock(uint256 blockHash, Block block, int peerId)
Parameters
Type Name Description
uint256 blockHash

The block hash.
Block block

The block.
System.Int32 peerId

ID of a peer that delivered a block.
Remarks

This method is called for all blocks that were delivered. It is possible that block that wasn't requested from that peer or from any peer at all is delivered, in that case the block will be ignored. It is possible that block was reassigned from a peer who delivered it later, in that case it will be ignored from this peer.
