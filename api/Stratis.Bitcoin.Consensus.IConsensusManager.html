GetOrDownloadBlocks(List<uint256>, OnBlockDownloadedCallback)

Parameters
Type Name Description
List<uint256> blockHashes

The block hashes to download.
OnBlockDownloadedCallback onBlockDownloadedCallback

The callback that will be called for each downloaded block.
Remarks

First we check if the block exists in chained header tree, then it check the block store and if it wasn't found there the block will be scheduled for download. Given callback is called when the block is obtained. If obtaining the block fails the callback will be called with null.
