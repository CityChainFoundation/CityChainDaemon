public class ProvenBlockHeaderStore : object, IProvenBlockHeaderStore, IProvenBlockHeaderProvider, IDisposable
Remarks

The pending batch is saved to the database and cleared every minute.

Items in the pending batch are also saved to the least recently used . Where the memory size is limited by Stratis.Bitcoin.Features.Consensus.ProvenBlockHeaders.ProvenBlockHeaderStore.MemoryCacheSizeLimitInBytes.

When new ProvenBlockHeader items are saved to the database - in case IProvenBlockHeaderRepository contains headers that are no longer a part of the best chain - they are overwritten or ignored.

When ProvenBlockHeaderStore is being initialized it will overwrite blocks that are not on the best chain.
