Starts an application defined task inside the async loop.

Declaration
IAsyncLoop Run(CancellationToken cancellation, TimeSpan? repeatEvery = null, TimeSpan? startAfter = null)
Parameters
Type Name Description
CancellationToken cancellation

Cancellation token that triggers when the task and the loop should be cancelled.
System.Nullable<TimeSpan> repeatEvery

Interval between each execution of the task. If this is , the task is only run once and there is no loop. If this is null, the task is repeated every 1 second by default.
System.Nullable<TimeSpan> startAfter

Delay before the first run of the task, or null if no startup delay is required.
