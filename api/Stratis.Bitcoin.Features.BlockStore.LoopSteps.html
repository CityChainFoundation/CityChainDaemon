Namespace Stratis.Bitcoin.Features.BlockStore.LoopSteps

Classes

BlockStoreInnerStep

Abstract class that all DownloadBlockSteps implement

BlockStoreInnerStepContext

Context for the inner steps, BlockStoreInnerStepFindBlocks and BlockStoreInnerStepReadBlocks.

The context also initializes the inner step InnerSteps.

BlockStoreInnerStepFindBlocks

Find blocks to download by asking the BlockPuller.

Find blocks until DownloadStack contains DownloadStackThreshold blocks.

If a stop condition is found Stratis.Bitcoin.Features.BlockStore.LoopSteps.BlockStoreInnerStepFindBlocks.ShouldStopFindingBlocksAsync(Stratis.Bitcoin.Features.BlockStore.LoopSteps.BlockStoreInnerStepContext) and there are still blocks to download, stop finding new blocks and only execute the read blocks inner step BlockStoreInnerStepReadBlocks.

BlockStoreInnerStepReadBlocks

Reads blocks from the BlockPuller in a loop and removes block from the DownloadStack.

If the block exists in the puller add the the downloaded block to the store to push to the repository. If Stratis.Bitcoin.Features.BlockStore.LoopSteps.BlockStoreInnerStepReadBlocks.ShouldBlocksBePushedToRepository(Stratis.Bitcoin.Features.BlockStore.LoopSteps.BlockStoreInnerStepContext) returns true, push the blocks in the Store to the block repository.

When the download stack is empty return a Stop result causing the BlockStoreLoop to start again.

If a block is stalled or lost to the downloader, start again after a threshold
