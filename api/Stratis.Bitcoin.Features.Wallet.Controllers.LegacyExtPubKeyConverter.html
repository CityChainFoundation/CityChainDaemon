Class LegacyExtPubKeyConverter

Stratis X has a version prefix starting xq5h. Full node has a version prefix xpub for extended public key. 3rd parties like Ledger haven't update yet to the full node version prefix. This class provides a way to convert from the legacy format to the current format.
