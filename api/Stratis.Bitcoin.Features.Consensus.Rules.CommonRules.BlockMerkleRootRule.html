Class BlockMerkleRootRule

This rule will validate that the calculated merkle tree matches the merkle root in the header.

public class BlockMerkleRootRule : IntegrityValidationConsensusRule, IIntegrityValidationConsensusRule, IConsensusRuleBase
Remarks

Transactions in a block are hashed together using SHA256 in to a merkle tree, the root of that tree is included in the block header.
