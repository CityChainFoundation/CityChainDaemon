Class CollateralPoAMiner

Collateral aware version of . At the block template creation it will check our own collateral at a commitment height which is calculated in a following way: counter chain height - maxReorgLength - AddressIndexer.SyncBuffer. Then commitment height is encoded in OP_RETURN output of a coinbase transaction.
