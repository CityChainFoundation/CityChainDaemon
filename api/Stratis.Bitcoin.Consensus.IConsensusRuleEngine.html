Interface IConsensusRuleEngine

An engine that enforce the execution and validation of consensus rule.

Namespace: Stratis.Bitcoin.Consensus
Assembly: Stratis.Bitcoin.dll
Syntax
public interface IConsensusRuleEngine : IDisposable
Remarks

In order for a block to be valid it has to successfully pass the rules checks. A block that is not valid will result in the Error as not null.
