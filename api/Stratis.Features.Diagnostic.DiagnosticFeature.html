Class DiagnosticFeature

Feature for diagnostic purpose that allow to have insights about internal details of the fullnode while it's running.

In order to collect internal details, this feature makes use of Signals to register to internal events published by the full node and uses reflection whenever it needs to access information not meant to be publicly exposed.

It exposes DiagnosticController, an API controller that allow to query for information using the API feature, when available.
