RunUntil(String, CancellationToken, Func<Boolean>, Action, Action<Exception>, TimeSpan)

Waits until a condition is met, then executes the action and completes.

Waiting is implemented using the async loop for which the task is defined as execution of the condition method followed by execution of the action if the condition is satisfied - i.e. returns true. If the condition is not satisfied, the loop waits as per setting and then it can be attempted again.
