Creates a snapshot based on difference of two performance counter snapshots.

Type Description
BackendPerformanceSnapshot

Snapshot of the difference between the two performance counter snapshots.
Remarks

The two snapshots should be taken from a single performance counter. Otherwise the start times of the snapshots will be different, which is not allowed.
