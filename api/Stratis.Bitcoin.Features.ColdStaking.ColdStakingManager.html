Class ColdStakingManager

The manager class for implementing cold staking as covered in more detail in the remarks of the ColdStakingFeature class. This class provides the methods used by the ColdStakingController which in turn provides the API methods for accessing this functionality.

Namespace: Stratis.Bitcoin.Features.ColdStaking

Syntax
public class ColdStakingManager : WalletManager, IWalletManager
Remarks

The following functionality is implemented in this class:

  • Generating cold staking address via the Stratis.Bitcoin.Features.ColdStaking.ColdStakingManager.GetFirstUnusedColdStakingAddress(System.String,System.Boolean) method. These addresses are used for generating the cold staking setup.
  • Creating a build context for generating the cold staking setup via the Stratis.Bitcoin.Features.ColdStaking.ColdStakingManager.GetColdStakingSetupTransaction(Stratis.Bitcoin.Features.Wallet.Interfaces.IWalletTransactionHandler,System.String,System.String,System.String,System.String,System.String,NBitcoin.Money,NBitcoin.Money) method.
