DefaultTransactionFee

The transaction fee used by the federation to build withdrawal transactions.

Declaration
public static readonly Money DefaultTransactionFee
Field Value
Type Description
Money
Remarks

Changing TransactionFee affects both the deposit threshold on this chain and the withdrawal transaction fee on this chain. This value shouldn't be different for the 2 pegged chain nodes or deposits could be extracted that don't have the amount required to cover the withdrawal fee on the other chain.

TODO: This should be configurable on the Network level in the future, but individual nodes shouldn't be tweaking it.
